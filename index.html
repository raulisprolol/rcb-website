Welcome to RCB

A powerful programming language combining R, C, and B

Features

Modern C Syntax

  • Pointer arithmetic and memory management
  • Structures and unions
  • Bitwise operations
  • Direct hardware access

R-style Data Analysis

  • Data frames and matrices
  • Statistical functions and distributions
  • Data visualization with ggplot2-style syntax
  • Time series analysis
  • Machine learning algorithms

OpenGL Graphics

  • Modern OpenGL 4.6 support
  • Shader programming
  • 3D modeling and rendering
  • Animation and physics integration

System Programming

  • Low-level system access
  • Memory management
  • Process control
  • Network programming
  • Device drivers

Package Management

Development Tools

Installation

Windows

1. Download the installer from our releases page:

https://github.com/rcb-language/rcb/releases/download/latest/rcb-windows-installer.exe

2. Run the installer and follow the on-screen instructions.

3. The installer will automatically add RCB to your PATH environment variable.

macOS

1. Install Homebrew if not already installed:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. Install RCB using Homebrew:

brew tap rcb-language/rcb
brew install rcb

3. Verify installation:

rcb --version

Linux

Ubuntu/Debian

sudo apt-get update
sudo apt-get install -y build-essential cmake libglfw3-dev libgl1-mesa-dev libglu1-mesa-dev

Fedora

sudo dnf install -y gcc make cmake glfw3-devel mesa-libGL-devel mesa-libGLU-devel

Arch Linux

sudo pacman -S --needed gcc make cmake glfw-x11 glu

Common Installation Steps

wget https://github.com/rcb-language/rcb/releases/download/latest/rcb-linux.tar.gz
tar -xzf rcb-linux.tar.gz
sudo cp rcb /usr/local/bin/

3. Verify installation:

rcb --version
